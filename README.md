# My Dunst config Dotfile

## Installation

Installation of it is pretty simple. First, if you already have a Dunst config directory, back it up:

```bash
mv ~/.config/dunst/ ~/.config/dunst_bak/
```
Then, clone my repo:

```bash
git clone https://github.com/GrumpyRumpus/dotfiles.git ~/
```

After that, move my dunst dotfiles to your .config/ directory:

```bash
mv ~/dotfiles/.config/dunst/ ~/.config/
```

Done :D
